# Boss Tracker 앱 디자인 아이디어

## 선택된 디자인 철학: 게임 UI 인스파이어드 대시보드

게임의 보스 추적 인터페이스를 모티브로 한 현대적이고 기능적인 디자인을 선택했습니다.

### 핵심 원칙
1. **명확한 상태 시각화**: 보스 출현/미출현 상태를 색상과 명암으로 즉시 인지 가능하게 표현
2. **게임스러운 미학**: 게임 UI 요소(원형 아이콘, 진행도, 상태 표시)를 현대적으로 해석
3. **직관적 상호작용**: 클릭 한 번으로 상태 변경, 이미지 수정 가능한 간단한 UX
4. **시각적 계층 구조**: 중요한 정보를 중앙에, 세부 조작은 우측에 배치

### 색상 철학
- **배경**: 어두운 톤 (게임 UI의 어두운 배경) - `oklch(0.141 0.005 285.823)`
- **활성 보스**: 밝은 색상 (황금색, 주황색, 보라색 등 다양한 색상)
- **비활성 보스**: 어두운 회색/검은색으로 표현
- **강조색**: 게임의 주요 색상 팔레트 사용

### 레이아웃 패러다임
- **좌측**: 보스 목록 (세로 스크롤 가능)
- **우측**: 선택된 보스의 상세 정보 및 이미지 수정 패널
- **상단**: 전체 상태 요약 (출현한 보스 수, 미출현 보스 수)

### 시그니처 요소
1. **원형 아이콘**: 게임 UI의 원형 보스 아이콘을 그대로 사용
2. **상태 표시 링**: 활성/비활성 상태를 테두리 색상으로 표현
3. **그라데이션 배경**: 보스 이미지에 미묘한 그라데이션 오버레이

### 상호작용 철학
- 보스 아이콘 클릭: 상태 토글 (활성 ↔ 비활성)
- 이미지 영역 클릭: 이미지 업로드/수정 모달 열기
- 마우스 호버: 보스 이름 및 상태 정보 표시

### 애니메이션
- 상태 변경 시: 부드러운 페이드 인/아웃 (300ms)
- 이미지 로드: 스켈레톤 로딩 상태
- 호버 효과: 아이콘 확대 및 그로우 이펙트

### 타이포그래피 시스템
- **제목**: Noto Sans KR Bold 700 (게임 UI의 임팩트 있는 제목)
- **본문**: Noto Sans KR Regular 400 (가독성 높은 기본 텍스트)
- **라벨**: Noto Sans KR Medium 500 (강조된 정보)
